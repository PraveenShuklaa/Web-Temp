=>HEADING TAGS

There are 6 types of heading tags as:-

  1. This is H1 tag!

    Used as:- <h1> Content </h1>

  2. This is H2 tag!

    Used as:- <h2> Content </h2>

  3. This is H3 tag!

    Used as:- <h3> Content </h3>

  4. This is H4 tag!

    Used as:- <h4> Content </h4>

  5. This is H5 tag!

    Used as:- <h5> Content </h5>

  6. This is H6 tag!

    Used as:- <h6> Content </h6>
